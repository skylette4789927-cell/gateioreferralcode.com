# Gate.io Referral Code Hub - 项目说明文档

## 1. 项目规划

### 1.1 项目简介
本项目名为 **Gate.io Referral Code Hub**，核心定位是帮助用户找到最新的 Gate.io 推荐码、注册奖励及使用教程。项目基于 Next.js 开发，支持多语言（英语、越南语、西班牙语、葡萄牙语、土耳其语、印尼语、俄语）。

### 1.2 技术栈
*   **框架**: Next.js (App Router)
*   **语言**: TypeScript
*   **样式**: Tailwind CSS
*   **多语言**: next-intl
*   **部署**: Vercel (预留)

### 1.3 页面结构
*   `/`: 首页 (重定向或作为着陆页)
*   `/gate-referral-code`: 核心着陆页 (SEO 重点)
*   `/gate-io-referral-bonus`: 奖励详情页
*   `/gate-io-invitation-code`: 邀请码页
*   `/gate-io-referral-code-reddit`: Reddit 评价页
*   `/gate-io-referral-id`: 教程页

### 1.4 多语言支持
*   默认: 英文 (en)
*   越南文 (vi)
*   西班牙语 (es)
*   葡萄牙语 (pt)
*   土耳其 (tr)
*   印尼语 (id)
*   俄语 (ru)
